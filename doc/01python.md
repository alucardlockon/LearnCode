### 列表
``` Python
l=[1,2]
l.append(3) #添加元素
l.insert(1,3) #在位置1添加3
l.count(1) #count个数
l.index(2) #返回2出现的第一个的位置
l.pop(1) #删除(返回删除的值)
l.remove(1) #删除
l.extend([3,4]) #在列表l中添加列表[3,4]
l.reverse() #反转列表
```
